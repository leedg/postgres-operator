# Security Policy

For the full security policy, vulnerability reporting process, supported
versions, and secure-operation recommendations, see
[docs/SECURITY.md](docs/SECURITY.md)
([한국어](docs/SECURITY.ko.md) · [日本語](docs/SECURITY.ja.md) · [中文](docs/SECURITY.zh.md)).

To report a vulnerability: **do not open a public issue.** Use
[GitHub Security Advisories](https://github.com/KeiaiLab/postgres-operator/security/advisories/new)
or email `security@keiailab.io`.
