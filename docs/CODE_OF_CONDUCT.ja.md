# Contributor Covenant 行動規範 (日本語)

> 英語原文: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) — canonical / 正本

本プロジェクトは [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を採用しています。

## 私たちの誓い

性別、性自認・表現、性的指向、障害、外見、体型、人種、民族、年齢、宗教、国籍を問わず、すべての人にハラスメントのない経験を提供することを誓います。

## 基準

ポジティブな環境を作る行動:

- 歓迎的で包摂的な言葉づかい。
- 異なる視点と経験への敬意。
- 建設的な批判を品位を持って受け入れる。
- コミュニティ全体にとって最善であることに集中する。
- 他のコミュニティメンバーへの共感。

容認されない行動:

- 性的表現・画像の使用、または望まない性的な注目。
- トローリング、侮辱的・軽蔑的なコメント、個人的・政治的攻撃。
- 公開または非公開のハラスメント。
- 明示的な許可なく他者の私的情報 (住所、メールアドレスなど) を公開すること。
- その他、専門的な環境において合理的に不適切と判断される行為。

## Enforcement (施行)

違反行為は `coc@keiailab.io` に報告できます。すべての報告は確認のうえ適切な対応がとられ、報告者の身元は秘匿されます。
