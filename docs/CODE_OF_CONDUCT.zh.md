# Contributor Covenant 行为准则 (中文)

> 英文原文: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) — canonical / 正本

本项目采用 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。

## 我们的承诺

无论性别、性别认同与表达、性取向、残疾、外貌、体型、种族、民族、年龄、宗教或国籍如何,我们承诺为所有人提供一个无骚扰的体验。

## 准则

营造积极环境的行为:

- 使用欢迎与包容的语言。
- 尊重不同的观点和经验。
- 优雅地接受建设性的批评。
- 关注对整个社区最有利的事情。
- 对其他社区成员表现出同理心。

不可容忍的行为:

- 使用性化语言或图像,或不受欢迎的性关注。
- 巨魔行为 (trolling)、侮辱或贬损的评论、个人或政治攻击。
- 公开或私下的骚扰。
- 未经明确许可,公开他人的私人信息 (地址、电子邮件等)。
- 其他在专业场合中可被合理认为不适当的行为。

## Enforcement (执行)

可向 `coc@keiailab.io` 报告违规行为。所有报告将被审查并采取适当措施,举报者的身份将保持保密。
